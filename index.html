Portfolio - Tokenomics

Pronti per la novità?

Token ERC20

Con ERC-20 si definisce lo standard grazie al quale possiamo creare dei token fungibili al di sopra di una Blockchain come quella di Etherem. Questi token hanno la caratteristica di essere fungibili quindi intercambiabili tra di loro, dando la possibilità di creare degli utility token (con un utilità all’interno di un network), security token (le possibili quote di un progetto) e governance token (per distribuire il consenso in una comunità).

Token ERC721

Con ERC-721 si definisce lo standard grazie al quale possiamo creare dei token non fungibili (NFT) che hanno la caratteristica di essere dei beni digitali scarsi e non replicabili! Questo standard ha aperto le porte a numero progetti ed ambiti di applicazione come collezionismo digitale (CryptoKitties), Virtual Worlds (Decentraland), Game Assets (Gods Unchained), CryptoArt fino alla tokenizzazione di beni reali e fisici! Vieni a scoprire il nuovo standard ERC-1155!
